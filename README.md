# Sistema de Control de Temperatura en un Edificio Inteligente

Este proyecto implementa un sistema que controla la temperatura en diferentes zonas de un edificio inteligente para optimizar el consumo de energía.

## Funciones implementadas

### leer_sensores_temperatura()
Simula la lectura de sensores de temperatura en diferentes zonas del edificio.

### calcular_temperatura_optima()
Calcula la temperatura ideal considerando:
- Hora del día
- Ocupación
- Temperatura exterior

### enviar_ajuste_climatizacion()
Envía señales al sistema de calefacción o refrigeración dependiendo de la temperatura actual.

### registrar_consumo_energia()
Registra el consumo energético y calcula el promedio para analizar la eficiencia del sistema.

## Beneficios
El uso de funciones permite:
- Mejor organización del código
- Reutilización de funciones
- Mejor mantenimiento del sistema
